# Taxi Translator

## Project Overview
Taxi Translator is a project designed to provide a simple and efficient tool for translating taxi-related phrases and information between different languages. This can be particularly useful for tourists and travelers who may not speak the local language, enabling them to communicate their needs effectively.

## Features
- Multi-language support for taxi-related phrases
- User-friendly interface
- Offline capabilities for quick access
- Ability to save favorite phrases
- Support for voice recognition and translation

## Quick Start Guide
1. Clone the repository:
   
   ```bash
   git clone https://github.com/gmxatzi02-sudo/Taxi-Translator.git
   ```

2. Navigate into the project directory:
   
   ```bash
   cd Taxi-Translator
   ```

3. Install the necessary dependencies:
   
   ```bash
   npm install
   ```

4. Run the application:
   
   ```bash
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to access the application.
